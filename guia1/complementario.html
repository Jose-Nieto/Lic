    • ¿Javascript no invasivo?

    El Javascript no invasivo, se fundamentan en separar la funcionalidad cliente de una página del contenido, ya que a fin de cuentas un documento HTML, no es otra cosa que un documento que nos permite difundir información en un medio (si no, preguntádselo a google) de manera que este, tiene que gozar de una estructura de marcación que nos permita conseguir ese fin.
    Así pues, el Javascript no invasivo, es aquel que está totalmente separado de la información-estructura sintáctica del sitio.
    Y el Javascript intrusivo, es aquel que está mezclado con la información estructura sintáctica de la página, considerándose como tal cualquier expresión de Javascript o sentencias ubicadas en el de nuestra página ya sean como atributos de un elemento o bloques de código.

    Datos Null y Undefined

    En javascript existen valores para las variables pero hay dos valores en especifico que se les puede otorgar a las variables los cuales muchas veces estas causa confusion entre los diferentes programadores ya que no conocen el significado correcto de este y son el NULL y el UNDEFINED.
    Pues bien empecemos por NULL:
    NULL para Javascript, la variable existe. En algún momento, explícitamente, la variable se estableció a null o esta variable esta vacia y no tiene nada guardada dentro.
    UNDIFINED:
    para Javascript, no existe. O bien no ha sido declarada o jamás se le asignó un valor. A direncia de NULL que menciona que si existe la variable pero no tiene nada dentro.
